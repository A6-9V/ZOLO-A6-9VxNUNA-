## Setup

This repository contains scripts to help you set up your EXNESS MetaTrader 5 environment and connect your MQL5 account.

### How to Run the Setup

1.  **Open PowerShell**: On your Windows VPS, open a PowerShell terminal. You can do this by searching for "PowerShell" in the Start Menu.
2.  **Navigate to the Repository**: Use the `cd` command to navigate to the directory where you have cloned this repository.
    ```powershell
    cd path\\to\\your\\repository
    ```
3.  **Run the Setup Script**: Execute the main setup script by running the following command:
    ```powershell
    ./Scripts/setup-exness-complete.ps1
    ```
4.  **Follow the On-Screen Instructions**: The script will guide you through the process of setting your MQL5 credentials and manually connecting your account in the MetaTrader 5 terminal.
